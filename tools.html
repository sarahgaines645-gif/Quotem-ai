What can Q do?

Q is a general AI you can trust — open-weights, frozen at a pinned version, and yours. Below is what he can do today. The list grows as he's trained.

In chat — just ask

Web search

Live, current results. Q searches when he doesn't already know — news, prices, dates, anything that may have changed.

"What's the current Bank of England base rate?"

Calculator

Real arithmetic, not LLM-flavoured guessing. Percentages, totals, VAT, currency conversions.

"What's 17.5% of 4283.50?"

Time & dates

Q knows the current time in any timezone — useful for scheduling, deadlines, and not getting "today" wrong.

"What time is it in Tokyo right now?"

Read documents

Drop a PDF, Word doc, image, or screenshot into the chat. Q reads it and answers questions about it. Scanned PDFs work too.

Drag a parking ticket in and ask "is this fair?"

Write a Word document

Letters, complaints, contracts, briefs. Ask Q for it and he'll hand back a real .docx you can download, save, or send.

"Write a complaint letter to the council about the bin collection."

Long-term memory

Tell Q something worth keeping and he remembers it across sessions. Names, preferences, projects, decisions.

"Remember my dad's birthday is March 15th."

APS — A Problem Shared

Toggle the APS button on. Hand Q a fine, bill, dispute, or letter you don't understand. He'll find the angle, write the appeal, and tell you the odds.

"I got a parking fine but the meter wasn't working."

Code & technical help

Write code, debug it, explain it. Q's good at most languages and frameworks; he'll be honest when he isn't sure.

"Why is this Python loop slow?"
